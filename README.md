
## Day 1 - Python 기초 학습 정리

### 📌 학습 목표
- Python 기본 입력 처리 패턴 이해  
- 문자열 및 숫자 처리  
- 리스트 입력  
- split / strip / map / list 변환 이해  
- 기본 실습 5문제 풀기

### ✔️ 핵심 문법
- `input().split()` : 공백 기준으로 문자열 분리  
- `list(map(int, ...))` : 문자열을 int로 변환 후 리스트로 생성  
- `a, b = ...` : 언패킹(unpacking)  
- `s[::-1]` : 문자열 뒤집기  
- `strip()` : 문자열 양쪽 공백 제거  
- `max(iterable)` : 비교 가능한 값들 중 최댓값 반환
